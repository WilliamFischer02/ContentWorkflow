# ContentWorkflow

A webpage for guiding the user step-by-step through their own content creation pipeline from start to finish, with progress tracking and redirect links.

## Live Site

[Open the live site](https://williamfischer02.github.io/ContentWorkflow/)

## About

ContentWorkflow is a React + Vite + TypeScript dashboard hosted on GitHub Pages.  
It is designed to visually resemble a descending production pipeline and help manage workflow stages across:

- Bingus The Wizard
- GOOB VISION
- GOOB Entertainment Co.

## Current Status

- GitHub Pages deployment is live
- Workflow dashboard UI is active
- Progress checkboxes and kanban memory are working in-browser
- CLI editor tooling is in progress
